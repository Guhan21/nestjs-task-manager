## Description

A To-Do API built using [Nest](https://github.com/nestjs/nest), TypeORM and PostgreSQL to demonstrate the core concepts of NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
